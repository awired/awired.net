Wiki Syntax

Wiki Formatzeichen können mit Hilfe eines Backslash "\" als Text im Eintrag verwendet werden..

Absatz

  • Eine Leerzeile zwischen den Absätzen lassen. ;
  • Paragraph : Text und eine Leerzeile ;
  • Titel : !!!, !!, ! je nach Ausführungsart (Wichtigkeit) ;
  • Horizontale Linie: ---- ;
  • Liste : Zeile mit einem * der # beginnen. Es ist möglich, die Listenelemente zu kombinieren. ;
  • Formatierter Text: Leerschlag vor jeder Textzeile ;
  • Blockquote/Absatz für Zitate : > oder ;: vor jeder Zeile mit Text. ;

Zeichen/Formatierungen

  • Die Satzzeichenkorrektur ist aktiviert. Alle Zeichen, welche keinen "zeilenumbrechenden" Charakter haben, werden durch Leerschlage ersetzt, sofern die Sprache = fr (Französisch) ist. Jedoch nur, falls diese Zeichen vorausgegangen sind:  ";","?",":" und "!". ;
  • Kursiv: zwei einfache Anführungszeichen ''Text'' ;
  • Fett und Kursiv: zwei Underscores  __Text__ ;
  • Neue Zeile: %%% ;
  • Einfügen: zwei Plus ++Text++ ;
  • Unterdrückung: zwei Minus --Text-- ;
  • Link: [URL], [Name|URL], [Name|URL|Sprache] oder [Name|URL|Sprache|Titel].
    Wenn man einen Link hinzufügen möchte, welcher auf einen Anker auf der gleichen Seite verweist, so muss dei URL mit dem #Ankername ersetzt werden. ;
  • Bild: ((URL|ALT Text)), ((URL|ALT Text|Position)) oder ((URL|ALT Text|Position|Beschreibung)).
    Der Parameter Position kann folgende Werte annehmen: L oder G (Links), R oder D (Rechts) oder C (Zentriert). ;
  • Anker: ~Anker~ ;
  • Akronym: ??Akronym?? oder ??Akronym|Titel?? ;
  • Zitate: {{Zitat}}, {{Zitat|Sprache}} oder {{Zitat|Sprache|URL}} ;
  • Code: @@Code@@ ;
  • Fussnote: $$Fussnote$$.
