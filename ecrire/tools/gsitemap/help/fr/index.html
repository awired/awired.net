Gsitemap

Présentation

Le plugin Gsitemap permet de créer un fichier XML Google Sitemaps afin de guider les robots d'indexation de Google sur votre site.

Ce plugin ne nécessite aucune modification du code principal de DotClear et peut être activé ou désactivé à volonté depuis son interface d'administration.

Les différentes options du plugin sont présentées ci-dessous.

Configuration

Activation du plugin

Vous pouvez librement choisir d'activer ou non la création du sitemap. Il suffit d'une case à cocher, la prise en compte est imméediate.

Lors de la désactivation, les autres paramètres de configuration restent en l'état et seront de nouveau appliqués à la ré-activation.

Configuration avancée

Elément(s) à intégrer

Vous pouvez choisir de faire figurer dans le fichiers sitemaps différents contenus issus de DotClear.

Ces éléments sont les suivants :

Page d'accueil
Indique si vous souhaitez faire figurer l'URI de la page d'accueil de votre blog.
Fils principaux
Indique si vous souhaitez faire figurer les URIs des fils de syndication principaux (fils RSS 1.0 et Atom 0.3 des billets).
Billets
Indique si vous souhaitez faire figurer les URIs de l'ensemble des billets du blog.
Catégories
Indique si vous souhaitez faire figurer les URIs des pages d'accueil des catégories.
Pages connexes
Indique si vous souhaitez faire figurer les URIs des différentes pages connexes. Vous devez disposer du plugin Pages connexes (Related) pour cela.

Réglages d'indexation

Priorité
Coefficient de pondération.
Il indique l'importance du contenu d'une page par rapport aux autres. Cette information n'est utilisée par Google que pour définir une priorité au moment de l'indexation.
Vous pouvez préciser une valeur numérique comprise entre 0.0 et 1.0.
Fréquence
Indique la fréquence à laquelle une page est susceptible de changer. Google prend en compte cette information pour plannifier l'indexation des pages, mais cela ne signifie pas qu'il "collera" à la fréquence.
Les valeurs possibles sont :
Non définie
Aucune fréquence n'est précisée.
Systématique
Signale que le contenu est susceptible de changer à chaque consultation.
Horaire
Le contenu peut changer d'heure en heure.
Journalière
Le contenu peut changer d'un jour à l'autre.
Hebdomadaire
Le contenu ne change que d'une semaine à l'autre.
Mensuelle
Le contenu ne change que d'un mois à l'autre.
Jamais
Le contenu est considéré comme simple archive, donc non sujet à modification.

Bouton "Pinguer Google"

En cas de modification importante dans la configuration de votre fichier sitemap, ce bouton vous permet de déclencher une requête HTTP pour re-soumettre automatiquement votre sitemap auprès de Google.
