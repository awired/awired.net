Préférences et gestion des utilisateurs

Pour modifier vos préférences, rendez vous sur l'onglet "préférences". Pour créer ou modifier un utilisateur, rendez vous sur l'onglet "rédacteurs". Vous devez être administrateur pour cette dernière action.

Champs

Prénom
Le prénom de l'utilisateur est facultatif.
Nom
Le nom de l'utilisateur est obligatoire.
Pseudo
Le pseudo de l'utilisateur est facultatif.

Note : Si l'utilisateur ne donne pas de pseudo, son nom et éventuellement son prénom seront utilisés pour indiquer qui est l'auteur du billet. S'il fournit un pseudo, alors ce dernier sera utilisé.

Email
L'addresse email de l'utilisateur est facultative. Si vous souhaitez activer la notification des commentaires de vos billets par email, vous devez renseigner ce champ.
Mot de passe
Choisissez un mot de passe assez long et peu explicite. En cas de mise à jour d'un utilisateur ou de vos préférences, laissez ce champ vide pour ne pas changer de mot de passe.
Langue
Si la langue existe dans les fichiers de traduction, elle sera utilisée pour l'interface d'administration de cet utilisateur. Chaque billet créé sera également "marqué" avec le code de cette langue automatiquement.
Format d'édition préféré
Le format choisi sera celui utilisé par défaut pour tout nouveau billet. L'utilisateur a bien sûr la possibilité de changer ce format ponctuellement pour un nouveau billet.
Statut de publication par défaut
Cette option définit le statut en ligne ou hors ligne de chaque nouveau billet.
Catégorie préférée
La catégorie choisie sera celle par défaut de chaque nouveau billet.
Hauteur de la zone d'édition d'un billet
Hauteur en lignes du champs d'écriture d'un billet.
Nombres d'heures entre l'heure de l'utilisateur et celle du serveur
Cette option permet de définir un éventuel décalage horaire entre votre serveur et votre lieu de publication. Les valeurs négatives sont possibles. L'heure de votre serveur est indiquée afin de définir votre décalage.

Administration

Ces champs ne sont disponibles que pour l'administrateur dans la zone de modification d'un rédacteur.

Identifiant
L'identifiant est le login de l'utilisateur. Il doit contenir uniquement des lettres et/ou des chiffres. Il est obligatoire.
Niveau de l'utilisateur
Le niveau correspond aux droits que vous donnez à l'utilisateur. Il existe trois types de droit :
  • Administrateur : tous les droits.
  • Rédacteur avancé : tous les droits sur ses billets, peut lire les autres billets, peut créer des catégories.
  • Rédacteur : tous les droits sur ses billets, ne peut pas lire les autres billets.
