Création des billets

Pour commencer la rédaction d'un billet, rendez vous dans la section billets puis cliquez sur nouveau billet.

Un billet doit comporter au moins un titre, et son contenu. Vous pouvez également choisir une catégorie.

Champs

Catégorie
La catégorie de votre billet. Pour créer une nouvelle catégorie, rendez vous dans la section catégories.
Format
Vous pouvez écrire vos billets dans deux formats ; Wiki ou HTML. Le format Wiki est une syntaxe simplifiée permettant de saisir rapidement un billet, avec un minimum d'apprentissage. Le format HTML vous donne, quant à lui, une latitude complète dans la mise en forme de vos écrits.
État
L'état permet de choisir si le billet est en ligne ou hors ligne après enregistrement.
Commentaires
Ce champ permet de définir si vous autorisez ou non les commentaires pour ce billets.
Trackbacks
Les trackbacks (ou rétroliens) sont des commentaires envoyés par un autre weblogue parlant de votre billet, ou du même sujet. Ce champ permet de définir si vous autorisez ou non les trackbacks.
Langue
Code langue de votre billet. Par défaut, il s'agit du code de votre langue. Vous pouvez le changer, par le code d'une autre langue, par exemple "en", "fr-qc". Ce code est libre, il sera utilisé lors de l'affichage des billets.
Sélection
Marquer un billet comme sélectionné peut permettre de créer une liste de certains billets importants que vous souhaitez mettre en avant, d'une manière ou d'une autre.
Titre
Le titre du billet proprement dit. Ce champ est obligatoire.
Titre URL
Le titre URL est la forme "URLisée" du titre. Les accents et les caractères génants pour une URL sont supprimés. Il est fait tout seul par le moteur sauf si vous en proposez un autre. Vous pouvez utiliser le lien "essayer" pour remplir le champ en fonction du titre de votre billet. Contrairement aux catégories, le titre du billet sous forme d'URL n'est pas obligatoire, simplement conseillé.
Chapô
La chapô permet d'écrire une courte introduction à un billet plus long dont la suite (le contenu) sera lisible en cliquant sur un lien.
Billet
Le corps de votre billet. Ce champ est obligatoire.
Insertion d'image
En cliquant sur "Insérer une image" vous ouvrirez une fenêtre permettant de choisir une image à insérer dans le corps de votre billet.
Bloc-notes
Le bloc-notes est une petite zone de texte permettant de conserver des notes, des liens ou tout ce qui vous passe par la tête pendant que vous rédigez votre billet. Ce champ n'est pas affiché sur votre weblog, il est là uniquement en cas de besoin.
Date et Heure
Vous pouvez changer la date et l'heure du billet telle qu'affichée sur votre blog.

Actions

visualiser
Permet de visualiser sommairement votre billet avant de l'enregistrer. Notez que si vous utilisez les émoticones, ils ne seront pas visibles sur cette partie.
Raccourci : alt+v.
enregistrer
Pour enregistrer ou mettre à jour votre billet.
Raccourci : alt+s.
supprimer
Supprime définitivement votre billet.

Ajout de commentaires

Vous pouvez ajouter un commentaire directement à un billet depuis l'interface d'administration. Seulement si le billet est enregistré.
