Einstellungen und Benutzermanagement

Um die Systemeinstellungen zu ändern muss das Register "Einstellungen" angewält werden. Sollen Benutzer (Redakture) erstellt oder geändert werden, so ist das Register "Redakteure" anzuwählen. Für letzteres müssen mindenstens Administratorrechte vorhanden sein.

Felder

Vorname
Der Vorname des Redakteurs (optional)
Name
Der Nachname des Redakteurs (optional)
Pseudonym
Das Pseudonym des Redakteurs (optional)

Beachte: Falls der Benutzer kein Pseudonym verwendet, wird der Vorname und Name im Weblog verwendet. Wird ein Pseudonym angegeben, so erscheint dieses im Weblog.

Email
Die Emailadresse des Redakteurs. Will der Redakteur über Kommentare auf seine Einträge benachrichtigt werden, so muss eine gültige Emailadresse erfasst sein.
Passwort
Das Passwort, um im Backend einzuloggen. Das Passwort sollte kryptisch gehalten werden, damit es sicher ist. Soll das Passwort nicht geändert werden, so muss dieses Feld leergelassen werden.
Sprache
Falls DotClear auf diese Sprache übersetzt worden ist, so wird die Administrationsoberfläche in der gewählten Sprache angezeigt. Daneben werden alle neuen Einträge mit diesem Sprachcode versehen.
Bevorzugtes Format
Definiert, welches Standardformat für die Erfassung der Einträge benutzt werden soll. Dies kann pro Eintrag überschrieben werden.
Bevorzugte Veröffentlichung
Diese Option erlaubt das Setzen eines Standardstatus für neue Einträge.
Bevorzugte Kategorie
Die für neue Einträge bevorzugte Kategorie.
Höhe des Textfelds für Beiträge
Höhe des Textfelds für Beiträge.
Differenz in Stunden zwischen Lokalzeit des Nutzers und Serverzeit
Diese Option erlaubt das Festlegen der Differenz zwischen der Lokalzeit und der Zeit auf dem Server.

Administration

Diese Felder sind nur sichtbar, falls der Benutzer Administratorenrechte besitzt.

Login-Name
Der Benutzername. Er darf nur aus Buchstaben und Ziffern bestehen. Das Feld ist obligatorisch.
Redaktionsrechte
Das Redaktionsrecht definiert die Privilegien für den entsprechenden Benutzer. Es gibt drei Typen von Rechten:
  • Administrator: Enthält alle Privilegien.
  • Redakteur mit besonderen Rechten: Das Recht erlaubt einem, seine eigenen Einträge zu administrieren, Einträge anderer Redakteure zu lesen und Kategorien zu erfassen.
  • Redakteur: Das Recht erlaubt einem, seine eigenen Einträge zu administrieren, jedoch keine anderen Beiträge zu lesen.
