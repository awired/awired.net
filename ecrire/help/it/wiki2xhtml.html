Sintassi Wiki

Se vuoi far sì che alcuni tag Wiki che hai inserito nel testo siano mostrati nel testo finale devi proteggerli con il tag "\".

Blocchi

  • Lascia una riga vuota tra ogni blocco dello stesso tipo ;
  • Paragrafo : per un nuovo paragrafo inserire una riga vuota ;
  • Titolo : !!!, !!, ! per titoli, ordinati per importanza descrescente ;
  • Linea orizzontale : ---- ;
  • Lista: all'inizio di ogni riga inserisci * o #. É possibile alternare le liste (*#*) in modo da generare liste innestate. Lo stile di ogni livello deve essere rispettato  ;
  • Testo preformattato : uno_spazio prima di ogni linea di testo ;
  • Blocco di quoting : > or ;: prima di ogni linea di testo.

Elementi

  • Il correttore automatico della punteggiatura è attivo se la lingua prescelta è fr (francese). Se attiva, tutti gli spazi che precedono uno di questi elementi : ";","?",":" and "!" verranno rimpiazzati da un unico spazio ;
  • Corsivo : ''testo'' ;
  • Grassetto : __testo__ ;
  • A capo : %%% ;
  • Sottolineato : ++testo++ ;
  • Barrato : --testo-- ;
  • Link : [indirizzo], [nome|indirizzo], [nome|indirizzo|lingua] or [nome|indirizzo|lingua|titolo].
    Se vuoi inserire un link ad un segnalibro presente nella pagina, metti come indirizzo #nome-del-segnalibro ;
  • Immagini : ((indirizzo|testo alternativo)), ((indirizzo|testo alternativo|posizionamento)) or ((indirizzo|testo alternativo|posizionamento|descrizione)).
    Valori per il campo posizionamento: L or G (sinistra), R or D (destra) or C (centro) ;
  • Segnalibro : ~nome-segnalibro~ ;
  • Acronimo : ??nome-acronimo?? or ??nome-acronimo|titolo?? ;
  • Citazione : {{citazione}}, {{citazione|lingua}} or {{citazione|lingua|indirizzo}} ;
  • Codice : @@codice@@ ;
  • Nota a piè di pagina : $$nota$$.
